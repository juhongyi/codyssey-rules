## Page 1

iMac macOS Sequoia 환경에서
한글 입력하기
아이맥에서 한글을 입력하기 위해서는 iMac 최초 로그인 시 구름 입력기 활성화를 해야 합니
다 .
( 자리 이동을 해도 최초 로그인 시도하는 iMac 이라면 해당 설정은 필수로 해야 합니다 .)
< 구름 입력기 활성화 >
iMac 최초 로그인 후 , [ 시스템 설정 (System Settings)] - [ 키보드 (Keyboard)]를 선
택합니다 .
[ 텍스트 입력 (Text Input)] 항목 [ 입력 소스 (Input Sources]] 우측의 [ 편집 ...
(Edit...)] 버튼을 누릅니다 .
좌측 하단의 [+] 버튼을 누르고 , [ 한국어 (Korean]] > [ 구름 (Han 2set)]을 선택한
뒤 [ 추가 (Add)]를 누릅니다 .
iMac macOS Sequoia 환 경 에 서 한 글 입 력 하 기

## Page 3

< 키 입력 수신 권한 승인 (Keystroke Receiving)>
로그인 후 “Gureum” 이 키 입력을 수신하고자 한다는 창이 뜨면 Deny 클릭합니다 .
팝업 내용 : "Gureum" would like to receive keystrokes from any application.
조치 : Deny 클릭
<macOS 한 / 영 전환 디폴트 단축키 >
Control + Space 눌러 한 / 영 전환합니다 .
입력 소스 전환 ( 한 / 영 전환 ): ⌃ (Control) + Space
<Spotlight 단축키 충돌 해결 >
만약 ⌃ (Control) + Space 단축키로 Spotlight 기능이 실행된다면 ,
[ 시스템 설정 (System Settings)] - [ 키보드 (Keyboard)] – [ 키보드 단축키 Keyboard
Shortcuts] - [Spotlight] 클릭하여 [ 기본값 복원 Restore Defaults] 로 Spotlight 의 단
축키를 변경합니다 .
iMac macOS Sequoia 환 경 에 서 한 글 입 력 하 기
